# Скриншоты

Клади сюда PNG — они используются в двух местах сразу:

1. **На GitHub** — вставь их в корневой `README.md` (см. раздел «Как выглядит»),
   тогда любой, кому ты дал ссылку на репозиторий, увидит превью прямо на главной странице.
2. **В установщике** — все `*.png` из этой папки автоматически вшиваются в
   `RevitQuickAccess-Setup.exe` при сборке и крутятся там автоматическим слайд-шоу.
   Ничего прописывать не надо: просто положил файл → `.\build.ps1` → он уже в установщике.

## Как назвать

Порядок показа — по алфавиту, поэтому удобно нумеровать:

```
01-binds.png
02-quick-commands.png
03-browser.png
04-transfer.png
05-mep-tools.png
```

## Рекомендации

- Ширина 1000–1400 px, PNG.
- Снимай саму панель плюс немного окружения Revit — так понятнее, что это плагин.
- 4–6 штук достаточно: слайд-шоу меняет кадр раз в 3.5 секунды.
